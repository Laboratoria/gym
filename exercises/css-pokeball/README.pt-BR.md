# CSS Pokebola

[https://cssbattle.dev/play/95](https://cssbattle.dev/play/95)

![Pokebola](css_pokeball.png)

__Objetivo__

Escreva o HTML/CSS no editor para replicar a imagem de destino à direita.

__Código base__

```html
    <div></div>
    <style>
      div {
        width: 100px;
        height: 100px;
        background: #dd6b4d;
      }
    </style>
```
