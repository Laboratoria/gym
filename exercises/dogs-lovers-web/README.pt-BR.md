# Dogs lover

__Objetivo:__

O objetivo é mostrar os dados dos filhotes na tela https://2k8tg.csb.app/
![img](https://user-images.githubusercontent.com/6140157/107087130-9b311900-67c0-11eb-8595-c6f73392f531.png)

## Considerações

A intenção neste exercício é poder exercitar os objetivos de aprendizagem:
>
>- [ ]Funciones
>- [ ]Arrays
>- [ ]Objects
>- [ ]Dom

## Os desafios

1) Consuma os dados dos filhotes

2) Mostre os filhotes e sua raça na tela

> __Importante__ ❗
>
> - O tempo máximo estimado para resolver este exercício é de 2 horas.
> - Não invista mais do que o tempo necessário neste exercício, lembre-se que é
> uma prioridade continue seu projeto de bootcamp.
>

> - Para este exercício os `data` dos filhotes estão no arquivo __data.js__
