# blackjack

[https://the-winter.github.io/codingjs/exercise.html?name=blackjack&title=Logic-2](https://the-winter.github.io/codingjs/exercise.html?name=blackjack&title=Logic-2)

Dado 2 valores int superiores a 0, devolver o valor
que estiver mais próximo de 21 sem passar por cima.
Devolver 0, se ambos passarem por cima.

__Exemplo__

```js
    blackjack(19, 21) → 21
    blackjack(21, 19) → 21
    blackjack(19, 22) → 19
```
